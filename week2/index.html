Chapter 3: Arrays, Logic, and loops

  • The pop, push, shift, and unshift can be used to modify an array and change its size. (rather than just deleating a part of it and leaving it as undefined).
      pop() : remove last item from the array.
      push() : appends a new value to the end of an array.
      shift() : remove the first item in an array.
      unshift() : appends a new calue to the begining of an array.
  • indexOf() : array method that returns the index of whatever is inside of the parenthesis.
  • includes() : array method that does a booleon check if whatever is inside of the parenthesis actually exists in the array.
  • maps are key : value pairs. Think JSON files.

Chapter 4: Functions

  • Arrow functions: Easier way to declare most functions
